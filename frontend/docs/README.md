# Skynex Consults Website

Production-ready consulting website built with:

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Supabase
- Resend

## Features
- Home, About, Services, Booking, Contact, Insights, Privacy Policy
- Booking form with email notifications
- Contact form with email notifications
- Admin login and dashboard
- DB-driven content structure
- SEO metadata, sitemap, robots
- Schema markup
- Turnstile support
- Optional Google Calendar support

## Setup

```bash
npm install
npm run check:env
npm run dev
